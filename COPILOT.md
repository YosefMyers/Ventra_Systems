# Copilot Prompt for Ventra Systems DTC Website Project

## Overview
This document contains the prompt utilized by Copilot for the Ventra Systems DTC website project. The purpose of this prompt is to guide the generation of appropriate content and code segments relevant to the functionalities and requirements of the DTC website.

## Copilot Prompt
"Create a responsive and user-friendly website for Ventra Systems' DTC that features the following:
1. Home page with a modern design that showcases the services provided.
2. Services page detailing each service with descriptions and images.
3. Contact page with a form for inquiries, integrated with email notifications.
4. About page outlining the company history and mission.
5. A blog section for updates and articles on topics related to DTC.
6. Responsive design that works on mobile, tablet, and desktop devices.
7. Incorporation of SEO best practices to enhance visibility in search engines."
